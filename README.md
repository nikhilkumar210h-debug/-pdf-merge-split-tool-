# 📄 PDF Tool – Merge & Split

A clean and beginner-friendly Python tool to **merge multiple PDFs** and **split PDF files** with a simple file dialog interface.

---

## 🚀 Features

✨ Merge multiple PDF files into one  
✂️ Split PDF (first 2 pages & last 2 pages)  
🖱️ Easy file selection using Tkinter dialog  
⚡ Fast and lightweight  

---

## 🛠️ Tech Stack

- Python 3
- pypdf
- Tkinter (built-in GUI)

---

## 📦 Installation

Clone the repository:
```bash

git clone https://github.com/nikhilkumar210h-debug/-pdf-merge-split-tool-
cd pdf-tool
```
Install dependencies:
pip install pypdf

________________________________________
## ▶️ Usage
Run the script:
python main.py
Then choose:
•	1 → Merge PDFs 
•	2 → Split PDF 
•	0 → Exit
